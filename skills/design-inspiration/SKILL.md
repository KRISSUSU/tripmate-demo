```
选一个风格方向：

① 极简黑白
   冷静、克制、高级感。Apple / Linear / Vercel 那种。
   大量留白，字体主导，几乎没有装饰。

② 小清新温暖
   奶油白底、莫兰迪色、圆角卡片。
   适合生活方式、食品、女性向产品。看着舒服不刺眼。

③ 大胆科技感
   深色渐变、霓虹光效、粒子背景。
   AI / Web3 / SaaS 工具常见气质。视觉冲击强。

④ 自然有机感
   绿色系、质感纸张、手写元素。
   环保、健康、户外、度假类产品。有温度、不冰冷。

⑤ 重势奢华感
   全屏大图、极少文字、金色点缀。
   高端酒店、时尚品牌、奢侈品气质。慢节奏、有分量。

⑥ 活力撞色
   高饱和撞色、不对称布局、大胆排版。
   年轻品牌、运动、潮流类产品。能量感强。

⑦ 数据极客风
   深色底、等宽字体、代码感界面、荧光绿/蓝高亮。
   开发工具、监控台、数据产品。

⑧ 特殊主题 🎯
   你说主题，我定制一套。
   例：中秋 / 复古港风 / Y2K / 暗黑童话 / 赛博朋克

回复数字，我给你出资源清单 + 关键词 + Demo。
```

**一句话选型：**
- Vibe Coding做页面 → **Heroicons** 或 **Phosphor**（简洁+Tailwind配套）

- 需要动态图标 → **Lottie**

### 🎬 动效资源

> **核心原则：先确定表达目标，再搭信息结构，先保证清楚，再追求好看。动效服务内容，不是越多越好。**

| 资源 | 地址 | 核心用途 | 使用原则 |
|---|---|---|---|
| React Bits | https://www.reactbits.dev | 文字动效、背景光效、卡片 hover | 挑 1-2 个亮点用，不要整页堆 |
| Motion Primitives | https://motion-primitives.com | 微交互 UI Kit，切换/展开/列表 | SaaS 功能区首选，克制自然 |
| Aceternity UI | https://ui.aceternity.com | Hero 区背景组件，粒子/光束/spotlight | 抠出 1 个背景效果，不要全套照搬 |
| Framer Motion | https://motion.dev | 理解动效概念和关键词 | 看 Examples 页，用关键词告诉 AI |
| GSAP | https://gsap.com | 复杂滚动叙事、时间线、SVG 动画 | 先学 ScrollTrigger 概念 |
| Animate UI | https://animate-ui.com | 按钮/弹窗/加载细节补充 | 配合 shadcn/ui，做最后抛光 |
| GSAP Showcase | https://gsap.com/showcase | 训练审美，看真实高级案例 | 带问题看：动效在帮内容还是干扰？ |
