# 🎨 Design Inspiration — 设计灵感导航 Skill

> 整合 16 个顶级设计资源站，一句话触发，秒出风格方案 + 参考链接 + 动效关键词。

---

## 能做什么

**触发一次，你会得到：**
- ✅ 8 种风格任选，精准匹配你的产品调性
- ✅ 对应参考网站 + 筛选关键词
- ✅ 推荐动效库 + 告诉 AI 用什么效果的关键词
- ✅ 可以直接说「帮我做个 Demo」，AI 立即开工

---

## 怎么用

说任意一句触发词即可：

> 「找设计灵感」「帮我找落地页参考」「做官网找风格」「找动效参考」

AI 会立刻弹出风格菜单，回复数字就行：

```
① 极简黑白   — Apple / Linear 那种，留白+字体主导
② 小清新温暖  — 奶油白底、莫兰迪色、圆角卡片
③ 大胆科技感  — 深色渐变、霓虹光效、粒子背景
④ 自然有机感  — 绿色系、质感纸张、有温度
⑤ 重势奢华感  — 全屏大图、金色点缀、慢节奏
⑥ 活力撞色   — 高饱和、不对称、能量感强
⑦ 数据极客风  — 深色底、等宽字体、荧光高亮
⑧ 特殊主题   — 你说主题，AI 定制（如：复古港风/Y2K/暗黑童话）
```
